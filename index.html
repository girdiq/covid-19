¿Qué herramientas informáticas utilizarías para ayudar a determinar posibles sitios de contagio masivo?

Una aplicación móvil con la API de google maps para determinar las zonas con más riesgo debido a la concurrencia de cada área

¿Cómo estructurarías una base de datos para el seguimiento de pacientes sospechosos de contagio por COVID-19?

La base de datos contendrá los datos generales de la persona: Nombre, edad, sexo etc. Además tendria su domicilio y los lugares por donde suele transitar y por último la fecha en la que empezó a presentar los síntomas

¿De qué manera pudieras hacer accesible y editable la base de datos propuesta en la actividad anterior, desde teléfonos móviles por parte del personal de salud que atiende a la población?

Mediante una aplicación que le permita a todo el personal capacitado agregar nuevos casos sospechosos con ayuda de un CRUD utilizando un web service con la API de REST y json

¿Qué herramientas tecnológicas utilizarías para detectar posibles portadores del virus en una estación del Metro?

Termómetros infrarrojos que permitan tomar la temperatura de las personas rápidamente y si es que presentan fiebre pasarlas a un test completo para conocer si es que son portadores del virus

¿Qué dispositivos emplearías para que los responsables de filtros sanitarios reciban información útil en tiempo real para dar seguimiento y atención a los posibles casos de riesgo de contagio, desde las herramientas que propusiste en la primera pregunta y cómo harías que se comunicaran?

Usaría una herramienta hecha con Arduino que se conectara al termómetro y al mismo tiempo a un dispositivo móvil para que a la hora que el termómetro tome la temperatura esta se comunique con el arduino y este a su vez con el móvil y así desde el celular capturar la información a la base de datos para que todos tengan acceso de inmediato

¿Qué elementos debe tener una interfaz gráfica de una app, para que al personal de filtros sanitarios le proporcione información útil sobre posibles personas infectadas por el COVID-19, y que información desplegaría cada uno de los elementos?

La interfaz debe contar con un menú desplegable donde contenga los siguientes apartados:

1. Apartado de áreas propensas a ocasionar contagios: En este apartado mostrará las áreas más concurridas en tiempo real

2. Apartado de sospechoso según su temperatura: Aqui se mostrara las personas que podrían ser portadoras del virus según la medida de temperatura que les fue tomada, además mostrará su es que ya se les ha hecho el test completo y de ser así la hora exacta en la que se le realizó

3. Apartado de personas contagiadas: Se mostrará los datos de las personas que se les detectó el virus

4. Apartado de Ingresar un nuevo caso confirmado: Se despliega un formulario para ingresar los datos de la persona infectada
